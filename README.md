# NIDALoc
NIDALoc: Neurobiologically Inspired Deep LiDAR Localization

## Environment

- python 3.7

- pytorch 1.9.1


## Data

We support the Oxford Radar RobotCar and NCLT datasets right now.


## Run
### Oxford

- train  -- 2 GPUs
```
python test.py
```

### NCLT

- train  -- 2 GPUs
```
python train.py
```
